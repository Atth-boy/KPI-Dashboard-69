ต้องการสร้างเว็บไซต์สำหรับแสดงข้อมูล dashboard งบลงทุนของฝ่ายควบคุมระบบไฟฟ้า
โดยอ้างอิงข้อมูลจาก google sheet แต่จะรันโปรแกรมผ่าน github
อ้างอิง base ข้อมูลตามไฟล์ excel "เก็บข้อมูลค่าใช้จ่ายโครงการ กสฟ." โดยไฟล์จริงเป็น google sheet 

อธิบายการเพิ่มข้อมูลในปัจจุบัน
ใน tab กรอกรายงานงบประมาณ - พนักงานจะอัพเดทข้อมูลค่าใช้จ่ายแต่ละเดือนของแต่ละโครงการ (เลขสัญญา วันเริ่มและสิ้นสุด fix ไว้อยู่แล้ว)
ใน tab กรอกเปอร์เซ็นงาน - เอาไว้กรอก % งานแต่ละเดือน แต่ละโครงการ

โดยตัวอย่าง dashboard ปีที่แล้วภาพในไฟล์ / กับร่างของปีนี้ (แต่ตอนนี้อยากเปลี่ยนเป็นแบบเว็บไซต์)

1.โดยจะมีการโชว์ เป็นกราฟหลักเป้าจ่ายแต่ละเดือนของงบทั้งหมดเทียบกับจ่ายจริงสะสมแต่ละเดือน (เป็นกราฟ s -curve)
2.กราฟรอง - เป้าจ่ายแต่ละเดือนของงบลงทุนโครงการเทียบกับจ่ายจริงสะสมแต่ละเดือนของงบลงทุนโครงการ (เป็นกราฟ s -curve)
3.กราฟรอง - เป้าจ่ายแต่ละเดือนของงบลงทุนปกติเทียบกับจ่ายจริงสะสมแต่ละเดือนของงบลงทุนปกติ (เป็นกราฟ s -curve)
*ประเภทงบลงทุนดูได้ที่รายละเอียดโครงการ
4.สามารถดูรายละเอียดแต่ละโครงการได้ โดยเลือกชื่อโครงการ จะแสดง ประเภท/งบ/เลข WBS/เป้าจ่ายรวม/เลขที่สัญญา/วันเริ่มและวันสิ้นสุดสัญญา
5.สามารถเช็คเป้าจ่ายของแต่ละโครงการ เป็นรายเดือน (สะสม) เทียบกับจ่ายจริงได้ ไม่ต้องเป็นกราฟก็ได้ เอาให้เห็นว่าเหมาะสม 
6.สามารถแสดง % ความคืบหน้างาน และรายละเอียดความคืบหน้า (อันนี้ยังไม่มีแต่เดี๋ยวค่อยทำกรอกในอนาคต)
